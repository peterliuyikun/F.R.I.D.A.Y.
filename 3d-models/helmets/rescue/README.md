# Rescue (Pepper Potts) Helmet

## Design Specifications

### Overview
Rescue suit helmet - sleek feminine design with elegant curves and refined proportions. Blue/silver color scheme with graceful aesthetics.

### Key Design Elements

**Elegant Cylindrical Profile:**
- Smaller scale (92% of standard)
- Smooth superellipsoid (n=2.5-2.6)
- Subtle crown detail
- Graceful proportions

**Refined Faceplate:**
- Sleeker, more delicate design
- Higher n values for smoother curves
- Subtle cheek accents
- Delicate brow line

**Sleek Jaw:**
- Tapered profile
- Refined chin detail
- Smooth transitions

**Elegant Ear Pieces:**
- Compact housing (16mm radius)
- Elegant ring detail
- Subtle dot pattern (4 positions)

**Technical Specs:**
- Smaller overall scale
- Higher n values for smoothness
- Target triangles: 12,000-22,000
- Scale: 92% of life-size
- Blue/silver color scheme

### 3D Printing Notes
- Smaller size prints faster
- Smooth curves print well with fine layers
- 0.15mm layer height recommended
- Supports minimal due to smooth design
