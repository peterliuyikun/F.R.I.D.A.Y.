# Iron Man Mark 42 Helmet

## Design Specifications

### Overview
The Mark 42 "Prehensile" suit helmet from Iron Man 3. Features a sleeker, more modular design with visible panel segmentation and a distinctive gold/silver color scheme.

### Key Design Elements

**Sleek Cylindrical Profile:**
- Narrower than Mark 3 (88% vs 92% of head radius)
- Lower superellipsoid exponent (n=2.0) for sharper edges
- More aerodynamic appearance

**Segmented Faceplate:**
- Visible panel lines (center vertical, horizontal brow)
- Modular assembly aesthetic
- Flatter front profile

**Compact Ear Pieces:**
- Reduced size (18mm vs 22mm radius)
- Integrated vent lines
- Sleeker profile

**Technical Specs:**
- Wall thickness: 2.5mm
- Target triangles: 15,000-25,000
- Scale: Life-size (580mm head circumference)

### 3D Printing Notes
- Print with supports for panel line details
- Consider multi-material for gold/silver contrast
- 0.15-0.2mm layer height recommended
