# Stealth Mode Helmet

## Design Specifications

### Overview
Stealth mode helmet with matte black, angular, tactical design. Features aggressive styling with sharp angles and low visibility features.

### Key Design Elements

**Angular Cylindrical Profile:**
- Low superellipsoid exponents (n=1.6-1.7)
- Ridge spikes on crown
- Side angular plates

**Angular Faceplate:**
- Very sharp angles (n=1.5-1.8)
- Sharp brow ridge
- Angled cheek plates

**Sharp Jaw:**
- Angular profile
- Chin spike detail
- Aggressive styling

**Stealth Ear Pieces:**
- Hexagonal housing (6 sides)
- Slit vents (stealth style)
- Cone back detail

**Technical Specs:**
- Very low n values for sharpness
- Spike and angular details
- Target triangles: 15,000-25,000
- Scale: Life-size
- Matte black color scheme

### 3D Printing Notes
- Sharp angles may need careful printing
- Supports needed for spikes
- Matte filament recommended
- 0.15mm layer height for sharp edges
