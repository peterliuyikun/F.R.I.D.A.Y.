# Iron Man Mark 3 Helmet

## Design Specifications

### Overview
The Mark 3 helmet represents the classic Iron Man design from the first MCU film. Features a rounded cranium, mechanical jaw with visible articulation seams, and detailed ear pieces with vent details.

### Technical Details

**Scale:** Life-size (head circumference: 580mm)

**Wall Thickness:** 2.5mm (optimized for 3D printing)

**Triangle Count:** ~15,000-25,000 (depending on subdivision)

**Techniques Used:**
1. **Superellipsoid cranium** - Smooth NURBS-like surface with n=2.3 for slightly squared but organic shape
2. **Revolved jaw profile** - Parametric curve rotated for mechanical jaw shape
3. **Boolean difference** - Visor cutout using CSG operations
4. **Subdivision surfaces** - One iteration of smoothing for refined geometry
5. **Mirror symmetry** - Perfect bilateral symmetry for consistency

### Design Features

**Cranium:**
- Superellipsoid base (a=92mm, b=95mm, c=102mm)
- Slightly elongated for heroic proportions
- Hollow shell construction

**Faceplate:**
- Flattened superellipsoid front section
- Positioned at 75% of head radius depth
- Integrated with cranium via boolean union

**Jaw Mechanism:**
- Revolved profile with 11 control points
- Visible articulation lines on sides
- Mechanical hinge appearance

**Ear Pieces:**
- Cylindrical housing (22mm radius)
- Three horizontal vent slots
- Back detail cylinder for depth

**Visor:**
- Curved opening using spherical segment
- 70mm wide, 35mm tall
- Positioned for proper eye alignment

### 3D Printing Considerations

**Orientation:** Print with face down for best surface quality

**Supports:** Required for:
- Visor opening edges
- Ear piece overhangs
- Jaw underside

**Infill:** 15-20% recommended for strength/weight balance

**Layer Height:** 0.2mm for good detail resolution

**Material:** PLA or PETG recommended

### Customization

Edit the generator script to adjust:
- `wall_thickness` - Change shell thickness
- `head_radius` - Scale entire helmet
- Superellipsoid parameters (n1, n2) - Change roundness
- Profile points - Modify jaw shape
