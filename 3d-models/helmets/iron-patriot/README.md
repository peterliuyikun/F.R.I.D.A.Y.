# Iron Patriot Helmet

## Design Specifications

### Overview
Iron Patriot helmet with patriotic red/white/blue theme. Features star details, stripe patterns, and bold patriotic styling.

### Key Design Elements

**Patriotic Cylindrical Profile:**
- Standard proportions
- Star detail on crown
- Side stripe patterns

**Patriotic Faceplate:**
- Center stripe detail
- Side accent stripes
- Bold styling

**Bold Jaw:**
- Strong chin detail
- Bold proportions

**Patriotic Ear Pieces:**
- Star detail on each ear
- Ring accent
- Bold styling

**Technical Specs:**
- Custom star generation function
- Stripe pattern details
- Target triangles: 15,000-25,000
- Scale: Life-size
- Red/white/blue color scheme

### 3D Printing Notes
- Star details require fine resolution
- Print with supports for star overhangs
- Consider multi-color printing
- 0.15mm layer height for detail
