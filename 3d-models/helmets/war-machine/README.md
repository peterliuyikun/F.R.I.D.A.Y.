# War Machine Helmet

## Design Specifications

### Overview
War Machine military heavy armor helmet. Features angular, aggressive styling with heavy plating, weapon mounts, and tactical military aesthetic.

### Key Design Elements

**Armored Cylindrical Profile:**
- Larger scale (105% of standard)
- Angular superellipsoid (n=1.8, 1.9)
- Top plate ridge and side armor plates
- Bolt details on side plates

**Tactical Faceplate:**
- Aggressive angular design
- Brow ridge and cheek guards
- Lower n value for sharper edges

**Heavy Jaw:**
- Reinforced chin guard
- Side reinforcements
- More angular profile

**Weapon Mount Ear Pieces:**
- Larger housing (24mm radius)
- Rail mount system
- Three mount points
- Rear exhaust port

**Technical Specs:**
- Thicker walls (3.0mm vs 2.5mm)
- Target triangles: 18,000-30,000
- Scale: Life-size + 5%
- Dark grey/silver color scheme

### 3D Printing Notes
- Thicker walls for durability
- Print with higher infill (20-25%)
- Supports needed for mount points and plates
- Consider ABS or PETG for strength
