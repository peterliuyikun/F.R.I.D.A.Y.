# Classic Comic Iron Man Helmet

## Design Specifications

### Overview
Classic comic book Iron Man helmet with retro rounded design. Features very smooth curves, vintage styling, and classic red/yellow color scheme.

### Key Design Elements

**Retro Rounded Cylindrical Profile:**
- Very high superellipsoid exponents (n=3.0)
- Classic fin detail on crown
- Smooth organic shape

**Classic Faceplate:**
- Very rounded design
- Simple brow ridge
- Vintage aesthetic

**Rounded Jaw:**
- Smooth profile curves
- Rounded chin detail
- Organic transitions

**Simple Ear Pieces:**
- Rounded housing
- Simple ring detail
- Classic styling

**Technical Specs:**
- High n values for roundness
- Classic fin detail
- Target triangles: 12,000-20,000
- Scale: Life-size
- Classic red/yellow color scheme

### 3D Printing Notes
- Smooth curves print well
- Fin detail needs supports
- Classic aesthetic suitable for display
- 0.2mm layer height acceptable
